Clash for Android使用教程

快 速 上 手

Clash for Android 支持两种导入配置文档的方式：

  • URL （订阅）
  • 本地导入
    首先讲一下订阅。

没有Clash订阅？ 请移步 👉 传送门

复制Clash订阅

打开 Clash for Android 应用程序，点击配置。

请在新弹出的窗口中点击右上角 ＋ 按钮

然后选择 「URL」，在对应地方填写订阅地址并保存。

返回首页。点击开关，即可进行代理。会提示是否同意创建 VPN，请点击允许。

开启代理后，可以点击中间的代理选项卡，进入策略组面板，在这里可以切换节点。直接点击你想要的节点即可进行切换。

点击⚡图标可以进行延迟测试，测试结果将显示在节点名称右侧。

点击右上角的三个点按钮，可以进入更多设置：

  • 刷新订阅
  • 更改代理模式
  • 改变代理组（策略组）排序
  • 改变代理（节点）排序
  • 前缀合并（即节点名字前缀相同的进行归类显示）

语言设置

语言设置路径：Settings → Interface → Language → Simplified Chinese
设置 👉 界面 👉 语言 👉 简体中文

暗黑模式

Clash for Android 现已适配暗黑模式。
入口：设置 👉 界面 👉 暗黑模式
