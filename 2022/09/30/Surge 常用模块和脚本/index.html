TestFlight 版 5.20.0 (2433) 或更新版本

  1. 模块安装链接

Network-Info

一个基于 Surge 5 客户端展示网络信息面板工具

功能

  1. 蜂窝网络下显示
    - 蜂窝数据 ISP & LTE、SA、NSA 等类型
    - 本地 IPv4、IPv6 位址
    - 代理节点 ISP、位置
  2. Wi-Fi 下显示
    - 本地 IPv4、IPv6 地址
    - 路由器 IPv4、IPv6 地址
    - 代理节点 ISP、位置

How to use

1. 安装环境

需要安装 Surge 客户端 ，并解锁面板功能
最低支持 :

AppStore 商店版 4.9.3 或更新版本
TestFlight 测试版 4.11.0 (2014) 或更新版本

2. 模块安装链接

简体中文正式版 : https://raw.githubusercontent.com/Rabbit-Spec/Surge/Master/Module/Panel/Network-Info/Moore/Network-Info.sgmodule
繁体中文正式版 : https://raw.githubusercontent.com/Nebulosa-Cat/Surge/main/Panel/Network-Info/Network-Info.sgmodule

3. 安裝方式

打开 Surge -> 模块 -> 安装新模块… -> 复制上方链接地址进行安装 -> 完成!

4. 更新模块方式

请按照以下方式进行更新

模块更新 :

Surge -> 模块 -> 找到需要更新的模块 -> 右滑点击更新完成更新

更新外部资源 :

点击首页左上角打开 配置列表 页面 -> 点击外部资源找到需要更新的资源，右滑点击完成更新，或者点击全部更新等待资源更新完成

更新时间建议间隔 5 分钟以上 ，更新间隔时间过短可能因页面缓存导致更新失败
(检查更新是否完成) Surge -> 脚本 -> 编辑器 -> 載入需要更新的脚本 -> 检查代码是是否与网页最新版本相同

注意事項

  1. 运营商显示功能改使用 $network 实现，通过 MNC 代码进行查询，不再使用 API 网站查询以提高隐私性
  2. 欢迎通过 拉取请求 提交更多语言翻译版本 与 补充其他电信运营商资料 (MNC 代码)

贴吧签到

百度贴吧每日自动签到

功能

1.每日上午9:30自动签到，显示签到贴吧总数、成功数量、失败数量。
2.针对签到失败的贴吧，进行10次，每次间隔2秒的重试，可以极大提高签到成功率。

注意事项

Surge在获取Cookie时，需要在MITM中设置h2=false，即关闭 MITM over HTTP/2，否则会提示获取Cookie出现异常。

需要用MITM over HTTP/2的话，建议在获取Cookie时临时关闭，获取成功后再开启，或者等待客户端修复。
